# 回放格式

## 占位符列表

| 占位符                                | 说明                     | 示例输出                 |
| ------------------------------------- | ------------------------ | ------------------------ |
| `${utc}`                              | 开始时间，UTC 时区       | 2026-01-01T00:00:00.000Z |
| `${utc:yyyyMMddHHmmss}`               | 开始时间，UTC，指定格式  | 20260101000000           |
| `${utcend}`                           | 结束时间，UTC            | 2026-01-01T01:00:00.000Z |
| `${utcend:yyyyMMddHHmmss}`            | 结束时间，UTC，指定格式  | 20260101010000           |
| `${start}`                            | 开始时间                 | 2026-01-01T08:00:00.000Z |
| `${start:yyyyMMddHHmmss}`             | 开始时间，指定格式       | 20260101080000           |
| `${end}`                              | 结束时间                 | 2026-01-01T09:00:00.000Z |
| `${end:yyyyMMddHHmmss}`               | 结束时间，指定格式       | 20260101090000           |
| `${lutc}`                             | 当前时间，转换为 UTC     | 2026-01-01T02:00:00.000Z |
| `${lutc:yyyyMMddHHmmss}`              | 当前时间 UTC，指定格式   | 20260101020000           |
| `${now}`                              | 当前时间                 | 2026-01-01T10:00:00.000Z |
| `${now:yyyyMMddHHmmss}`               | 当前时间，指定格式       | 20260101100000           |
| `${(b)yyyyMMddHHmmss}`                | 开始时间 `(b)`，指定格式 | 20260101080000           |
| `${(e)yyyyMMddHHmmss}`                | 结束时间 `(e)`，指定格式 | 20260101090000           |
| `${(b)timestamp}`                     | 开始时间，毫秒时间戳     | 1767225600000            |
| `${(e)timestamp}`                     | 结束时间，毫秒时间戳     | 1767229200000            |
| `${(b)timestampSeconds}`              | 开始时间，秒时间戳       | 1767225600               |
| `${(e)timestampSeconds}`              | 结束时间，秒时间戳       | 1767229200               |
| `${duration}`                         | 时长（秒）               | 3600                     |
| `${(b)}`                              | 开始时间，默认毫秒时间戳 | 1767225600000            |
| `${(e)}`                              | 结束时间，默认毫秒时间戳 | 1767229200000            |
| `${(b)10}`                            | 开始时间，秒时间戳简写   | 1767225600               |
| `${(e)10}`                            | 结束时间，秒时间戳简写   | 1767229200               |
| `${(b)yyyyMMddHHmmss\|Asia/Shanghai}` | 开始时间，上海时区       | 20260101080000           |
| `${(e)yyyyMMddHHmmss\|Asia/Shanghai}` | 结束时间，上海时区       | 20260101090000           |
| `${(b)yyyyMMddHHmmss\|UTC}`           | 开始时间，UTC 时区       | 20260101000000           |
| `${(e)yyyyMMddHHmmss\|UTC}`           | 结束时间，UTC 时区       | 20260101010000           |
| `${(b)yyyyMMddHHmmss:utc}`            | 开始时间，UTC 快捷写法   | 20260101000000           |
| `${(e)yyyyMMddHHmmss:utc}`            | 结束时间，UTC 快捷写法   | 20260101010000           |
| `${start\|yyyyMMddHHmmss}`            | 开始时间，指定格式       | 20260101080000           |
| `${end\|yyyyMMddHHmmss}`              | 结束时间，指定格式       | 20260101090000           |
| `{utc}`                               | UTC 开始时间（短语法）   | 2026-01-01T00:00:00.000Z |
| `{utc:YmdHMS}`                        | UTC 开始时间，短格式     | 20260101000000           |
| `{utcend}`                            | UTC 结束时间             | 2026-01-01T01:00:00.000Z |
| `{utcend:YmdHMS}`                     | UTC 结束时间，短格式     | 20260101010000           |
| `{start}`                             | 开始时间                 | 2026-01-01T08:00:00.000Z |
| `{start:YmdHMS}`                      | 开始时间，短格式         | 20260101080000           |
| `{end}`                               | 结束时间                 | 2026-01-01T09:00:00.000Z |
| `{end:YmdHMS}`                        | 结束时间，短格式         | 20260101090000           |
| `{lutc}`                              | 当前时间 UTC             | 2026-01-01T02:00:00.000Z |
| `{lutc:YmdHMS}`                       | 当前时间 UTC，短格式     | 20260101020000           |
| `{now}`                               | 当前时间                 | 2026-01-01T10:00:00.000Z |
| `{now:YmdHMS}`                        | 当前时间，短格式         | 20260101100000           |
| `{(b)YmdHMS}`                         | 开始时间 `(b)`，短格式   | 20260101080000           |
| `{(e)YmdHMS}`                         | 结束时间 `(e)`，短格式   | 20260101090000           |
| `{(b)timestamp}`                      | 开始时间，毫秒时间戳     | 1767225600000            |
| `{(e)timestamp}`                      | 结束时间，毫秒时间戳     | 1767229200000            |
| `{(b)timestampSeconds}`               | 开始时间，秒时间戳       | 1767225600               |
| `{(e)timestampSeconds}`               | 结束时间，秒时间戳       | 1767229200               |
| `{duration}`                          | 时长（秒）               | 3600                     |
