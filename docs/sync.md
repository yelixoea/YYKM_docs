# 数据同步

用于将应用中的配置数据同步到其他设备或服务中，或从其他设备或服务中拉取数据到应用中。

## 支持协议

| 协议         | 支持拉取 | 支持推送 |
| ------------ | -------- | -------- |
| WebDAV       | ✅        | ✅        |
| HTTP / HTTPS | ✅        | ❌        |
| File         | ✅        | ✅        |
| FTP          | ✅        | ✅        |
| SMB          | ✅        | ✅        |
| Gitee Gist   | ✅        | ✅        |
| GitHub Gist  | ✅        | ✅        |
| 其他设备     | ✅        | ✅        |

## 其他设备

通过局域网或互联网访问其他设备上的资源进行同步备份。

**URI 格式**

```
http://<device-ip-or-domain>:10481/api/sync/data
```
