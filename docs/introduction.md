# 介绍

云影空蒙是一款基于 **Android 原生开发** 的电视直播空壳软件。本应用**不包含任何直播源**，用户需要自行添加直播源才能观看电视节目。

## 软件功能

- 开机自启
- 画中画
- 国际化
- 多订阅源
- WebDav、SMB、FTP
- DIYP、超级直播节目单
- 频道预览
- 频道收藏
- 频道搜索
- 节目回放
- 换台反转
- 数字选台
- 音频软解
- 记忆播放进度
- 记忆解码配置
- Media3、IjkPlayer、Mpv 播放器
- 自定义 UA、Header、代理等参数
- 字幕
- 主题、壁纸、夜间模式
- Tbs X5 内核
- 数据同步
- 二进制
- 多屏同播
- 远程推送
